# Personal Resume Landing Page

Landing page estática que funciona como CV online.

## Objetivo
Mostrar de forma clara y profesional mi perfil como QA Automation Engineer,
con foco en automatización, Python y sistemas críticos.

## Tecnologías
- HTML5 semántico
- CSS3
- JavaScript
- GitHub Pages (deploy sugerido)

## Estructura
- `index.html`: página principal
- `assets/css`: estilos
- `assets/js`: scripts
- `assets/img`: imágenes

## Deploy
El proyecto puede publicarse fácilmente usando GitHub Pages.

## Evolución futura
Este sitio puede migrarse a una versión dinámica utilizando Python (Flask o FastAPI)
y templates Jinja2, leyendo los datos del CV desde archivos JSON o YAML.
